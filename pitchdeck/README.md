# pitchdeck/

LeanScan is indie / not VC-funded (per context doc § 2). So this folder is **not** for fundraising decks.

## What it's actually for

- `influencer-pitch.pdf` — the deck you send to micro-influencers explaining the founder-cohort trade (free lifetime access in exchange for honest feedback).
- `rd-partnership-pitch.pdf` — the version for registered dietitians who specialize in GLP-1 nutrition.
- `press-kit.pdf` — when a journalist or podcaster asks "tell me about LeanScan," send this.
- `b2b-explainer.pdf` — far future. Clinics, telehealth GLP-1 providers (Hims, Ro, WeightWatchers GLP) might license LeanScan as a companion. Not a 2026 problem.

## Build order

Do nothing here until you have:
1. 500+ waitlist signups (proof the niche exists)
2. The shipping MVP (proof you can execute)
3. A clear ask of the recipient (free access → honest review)

Decks built earlier are wasted effort.
