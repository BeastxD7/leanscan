# research/

Primary research — the raw evidence that the niche, pain, and positioning are real.

```
research/
├── competitor-reviews/    Mining negative reviews of MyFitnessPal, Cal AI, Lose It
└── user-interviews/       Transcripts and notes from real GLP-1 users
```

## competitor-reviews/

Holds `review-mining.xlsx` — the days 3–4 spreadsheet. 25 seeded complaint patterns + 75 blank rows + auto-counting summary. Target: 100+ real verbatim quotes by end of day 4.

## user-interviews/

Empty for now. Add when you start talking to GLP-1 users directly. Format:

```
user-interviews/
├── 2026-05-15-sarah-w-wegovy.md
├── 2026-05-18-mike-m-mounjaro.md
└── synthesis.md     Patterns that emerge across interviews
```

Use the founder-cohort waitlist as a recruiting pool — every new signup is a potential 15-minute Zoom call. Aim for 10 interviews before you finalize the v1 spec.

## What does NOT go here

- Generic market sizing → `../market/sizing/`
- Competitor feature comparisons → `../market/competitors/`

`research/` is primary (you collected it yourself). `market/` is secondary (you read it somewhere).
